## Ocvrm

OpenCV in RoboMaster. colorArea, lineFollow and ballTrack.

## Enviroment

### Software

- QtCreator 4.8.2
- GCC 8.3.0
- OpenCV 4.4.0
- wiringPi 2.52

### Hardware

- Raspberry Pi 4B
